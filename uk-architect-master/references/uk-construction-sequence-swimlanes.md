# UK construction sequence — swimlane archetypes

Illustrative sequences only. Project-specific programme required for binding commitments.

## RC frame tower (typical)

| Phase | Trades | Hold points |
|-------|--------|-------------|
| Mobilisation | Hoarding, piling, foundations | BC foundation inspection |
| Core / frame | RC frame, core climb | Frame survey; fire stopping at slabs |
| Envelope | Follow-the-structure facade, roofing | Watertight milestone |
| Fit-out | MEP first fix, partitions, finishes | Pre-commissioning |
| Commissioning | Services, fire, lifts | BC final inspection |

**Typical floor cycle:** 5–10 days per floor (project-specific).

## Steel frame + cladding

| Phase | Notes |
|-------|-------|
| Steel erection | Crane logistics; fire protection to steel |
| Metal deck / concrete topping | Acoustic and fire separation |
| Cladding | Early enclosure for dry trades |
| Fit-out | As RC tower |

## MMC / modular

| Phase | Notes |
|-------|-------|
| Module manufacture | Parallel to site civils (`uk-mic-dfma`) |
| Module delivery and install | Crane slots; highway permits (`uk-site-establishment`) |
| Connector / corridor fit-out | Weather-dependent |
| Commissioning | Module-level tests before connection |

## Interface rules

1. Architect issues IFC packages before contractor sequences affected zones.
2. Fire stopping inspected before concealment.
3. Commissioning precedes BC completion (`uk-op-submission-strategy`).
4. Early freeze: facade panel types, riser layouts, lift cores.
