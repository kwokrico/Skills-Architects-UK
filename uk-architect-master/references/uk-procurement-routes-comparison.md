# UK procurement routes — comparison matrix

Use with `uk-procurement-strategy` for route selection. Confirm client objectives before recommending.

| Criterion | Traditional (JCT Standard/Intermediate) | Design & Build (JCT D&B) | Management (JCT MC/CM) | NEC4 ECC |
|-----------|----------------------------------------|--------------------------|------------------------|----------|
| **Design responsibility** | Employer / architect to tender | Contractor (except ERs) | Employer via trade packages | Per option (A fixed, C target cost) |
| **Single point responsibility** | No (employer coordinates) | Yes | Manager coordinates trades | Shared via compensation events |
| **Price certainty at tender** | High (if design complete) | High (lump sum) | Lower until packages let | Option-dependent |
| **Programme risk** | Contractor | Contractor | Manager + trades | Shared |
| **Design quality control** | Employer retains | Employer via ERs | Employer retains | Employer retains |
| **Change mechanism** | Variations (JCT) | Employer change / CE | Trade package variations | Compensation events |
| **Weather delay** | Relevant event (JCT) | Relevant event | Trade-level | Compensation event (NEC) |
| **Best for** | Quality-led; complex design | Speed; employer accepts design transfer | Fast start; complex interfaces | Collaborative; public sector |
| **Typical public sector** | Less common | Common | Less common | Very common (NEC4) |
| **HRB considerations** | Full design control | Competency transfer to contractor | Employer retains design duty | Gateway evidence per dutyholder |

## Decision prompts

1. Is design complete enough to tender? If no → D&B or early contractor involvement.
2. Does employer need design control? If yes → traditional or management.
3. Is collaborative risk-sharing required? If yes → NEC4 target cost (Option C).
4. Is programme the primary driver? If yes → D&B or management with early packages.

## Stage-gate outputs

| Gate | Output |
|------|--------|
| Pre-tender | Route recommendation memo, contract form selection |
| Tender | Tender documents, pricing document, programme |
| Award | Signed contract, bonds, insurances, mobilisation plan |
