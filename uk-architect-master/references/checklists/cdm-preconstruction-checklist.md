# CDM 2015 — pre-construction checklist

Use with `uk-construction-health-safety`. Distinguish CDM Principal Designer from Building Regulations Principal Designer (BSA).

## Project classification

- [ ] Notifiable project? (≥30 days with 20+ workers, or ≥500 person-days)
- [ ] F10 notification submitted to HSE (if notifiable)
- [ ] Client CDM duties understood (competency, resources, information)

## Appointments

- [ ] Principal Designer appointed (design phase) — CDM reg. 11
- [ ] Principal Contractor appointed before construction — CDM reg. 12
- [ ] BR Principal Designer (BSA) appointed separately if HRB — `uk-building-codes`

## Pre-construction information

- [ ] Client provides existing building information (asbestos, structure, services)
- [ ] Design hazards identified and eliminated/reduced so far as reasonably practicable
- [ ] Residual risks documented for construction phase plan and H&S file
- [ ] Welfare facilities requirements communicated

## Design coordination

- [ ] Temporary works interfaces identified
- [ ] Coordination with structural, services, fire strategies
- [ ] Information issued to Principal Contractor before construction starts

## H&S file (planning)

- [ ] H&S file structure agreed
- [ ] Residual risk information schedule
- [ ] Handover to client at project end

## Halt rules

- Halt duty advice if notifiability unknown.
- Route fire strategy to `uk-fire-life-safety` — not CDM scope.
