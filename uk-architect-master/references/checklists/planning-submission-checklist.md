# Planning submission checklist (advisory)

Tailor to LPA validation requirements and application type.

## Core

- [ ] Application form and correct fee
- [ ] Ownership certificate and notices
- [ ] Site/plan and block plans (scale, north point)
- [ ] Existing and proposed drawings (elevations, sections, floor plans)
- [ ] Design & Access Statement (where required)
- [ ] Planning statement / supporting text
- [ ] CIL / s106 heads of terms awareness

## Technical annexes (as required)

- [ ] Transport statement / TRA
- [ ] Flood risk / drainage
- [ ] Ecology / arboriculture
- [ ] Heritage statement (listed/conservation)
- [ ] Fire statement (HRB / policy-driven)
