# BSA 2022 gateway checklist (HRB — advisory)

Use with project-specific legal and BSR guidance.

## Gateway 1 (planning)

- [ ] HRB classification confirmed
- [ ] BSR engaged per planning gateway rules
- [ ] Fire statement / safety case inputs aligned with planning submission
- [ ] Dutyholder intentions documented

## Gateway 2 (construction)

- [ ] Design meets Building Regulations functional requirements
- [ ] Golden-thread information structure defined (ISO 19650 alignment where used)
- [ ] Change control process for safety-critical design changes
- [ ] Building Control / BSR submission pack complete

## Gateway 3 (completion)

- [ ] As-built information and golden-thread completeness
- [ ] Regulation 38 / fire safety information pathway confirmed
- [ ] Resident / accountable person handover plan (where applicable)
